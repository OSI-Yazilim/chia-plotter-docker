# Chia Plotter powered by `docker-compose`

## Usage

- **Configure For Use Cases**
```bash
# For Linux or MacOS
mv .env.example .env

# For Windows in Powershell
ri .env.example .env

# For Windows in CMD
rename .env.example .env
```

- **Running**

```bash
docker-compose up
```

- **Stopping**
```bash
docker-compose down
```
